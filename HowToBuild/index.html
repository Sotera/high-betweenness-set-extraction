Requirements

  • Git
  • Gradle
  • Java 7
  • Apache Giraph 1.1.0-SNAPSHOT (Built for CDH 5.0.0)
  • CDH 5.0.0 (MRv1)

How To Build

This project depends on Apache Giraph, you'll need to have giraph built for the version of hadoop on your cluster, and installed into your local maven repository before you can build the project. For some help with building giraph see below.

Edit the build.gradle file to ensure the build uses your version of giraph and hadoop.

You can now build the project by running:

    
        $ gradle clean fatjar

Verify that build/libs/HighBetweennessSetExtraction-1.0.jar exists.
